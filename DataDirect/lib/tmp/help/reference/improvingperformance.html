8 Improving Performance

This topic provides information and techniques you can use to enhance the performance or your DataDirect XQuery applications. This chapter covers the following topics:

  • Querying Large XML Documents
  • Using Comparisons
  • Understanding Compensation
  • Using Query Pooling
  • Using Connection Pooling

If you work with relational databases, see Using DataDirect XQuery SQL Generation Algorithms to learn about different ways DataDirect XQuery can generate translations of XQuery code to SQL statements.
