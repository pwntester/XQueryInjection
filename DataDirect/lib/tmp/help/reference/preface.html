This help is your guide and reference to DataDirect XQuery® from DataDirect Technologies and describes how to use DataDirect XQuery to access and update both XML and relational sources, and to return XML results. This book provides information about the following topics:

  • Using DataDirect XQuery to query both XML and relational sources, and return XML results
  • Using DataDirect XQuery to update relational sources
  • DataDirect XQuery support for XQuery and the XQuery API for Java™ (XQJ)
  • Examples and tutorials that show how you can use DataDirect XQuery in your environment
  • Using DataDirect Spy™ for XQJ, a tracing and logging utility for troubleshooting
